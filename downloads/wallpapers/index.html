Foxkeh Wallpapers

Foxkeh's monthly wallpapers reflecting all four seasons in Japan. Choose from calendar or no-calendar versions for your desktop! Note that holidays are not included in these calendars.

How to set a wallpaper with Firefox

Click one of the links below. Right-click the image and choose Set As Desktop Background. In the preview dialog, click Set Desktop Background button.

If the wallpaper doesn't fit your desktop: On Windows, select Stretch from the Position drop-down menu. On Mac, click Open Desktop Preferences button to open System Preferences, then select "Stretch to full screen."
